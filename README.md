# GO-JOBBER

Go-Jobber is a job search Website which built on **REACT JS** and **MONGODB** and deploy on **Heroku**.

## DESCRIPTION
Go-Jobber website consists of the following pages:
1.	**Home**
2.	**About**
3.	**Contact**
4.	**Categories**
5.	**Job Search**
6.	**Job Post**
7.	**Profile**
8.	**Login / Signup**
9.	**Apply Job**
